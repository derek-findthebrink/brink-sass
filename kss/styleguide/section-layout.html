6.2.1@mixin grid-table #permalink

  • $row-el
    the element selected to function as a row, defaults to ul
  • $cells-el
    the element selected to function as a cell within a row, defaults to li
  • $pad
    internal padding to include on each cell, defaults to flex-gutter()
  • $max-width
    the maximum width of the table, defaults to 100%

grid with no spaces between cells. each cell will line up side by side within row, each equally spaced
